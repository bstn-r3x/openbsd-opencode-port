OpenCode for OpenBSD (ports local-bootstrap)

This package currently installs a locally built OpenCode binary through the
OpenBSD ports framework while the final distfile/source build strategy is being
implemented.

Installed paths:
- /usr/local/bin/opencode
- /usr/local/libexec/opencode/opencode-bin
- /usr/local/share/doc/opencode/README.txt
- /usr/local/share/doc/opencode/TROUBLESHOOTING.txt
